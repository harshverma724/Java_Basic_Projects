# Password Generator

A simple and customizable password generator written in Java. This program generates a random password based on user preferences, such as the inclusion of lowercase letters, uppercase letters, digits, and special characters.

## Features

- Generates random passwords with customizable length.
- Allows the user to choose character types to include in the password (lowercase, uppercase, digits, and special characters).
- Uses `SecureRandom` for stronger randomness and security.

## Requirements

- Java 8 or above
- Command-line interface for running the program
